Data formats

  • You already now that data can be formatted as string (text) or numerical.
    (String variables (text) require " " whenever you are referring to them)
  • Sometimes it can be useful to switch the format. You can do this by
. sencode name, replace // to change from string to numerical

or

. sdecode number, replace // to change from numeric to string
  • When you are working with time series, the time variable can be stored as date.
. list date in 1/5

     ┌───────────┐
     │      date │
     ├───────────┤
  1. │ 01jan2013 │
  2. │ 02jan2013 │
  3. │ 03jan2013 │
  4. │ 04jan2013 │
  5. │ 05jan2013 │
     └───────────┘

In this case it might be useful to extract the day, month or year. You can do this by using the following functions:

. generate    day   = dow(date)

. generate    month = month(date)

. generate    year  = year(date)

Working with time series data

Define the time variable. This is necessary for stata to know the structure of the dataset and will you allow various shortcuts.

. tsset  date
        time variable:  date, 01jan2013 to 31dec2016
                delta:  1 day

For example, for plotting graphs you can use tsline volume instead of line volume date

. tsline volume
